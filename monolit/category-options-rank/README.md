Скрипты позволяют перенести на новый сайт, с другими ID, rank опций на категории

Последовательность:

1. Получить опции с сайта оригинала get_category_options.sql. Сохранить результат как PHP массив
2. Обработать массив файлом data_to_json.php
3. Полученные данные JSON из output.json перенести в переменную update_options_rank.php
4. Запустить update_options_rank.php
